atomic-boxes⌗

Vagrant boxes with Atomic OS like VMware-Photon / CoreOS / Feroda-CoreOS

This repository contains files used by Packer to create images for Vagrant Cloud with Atomic OS

Table of Contents⌗

  1. Overview
  2. Requirements
  3. Building

Overview⌗

CoreOS provides Vagrant boxes for each release, in formats compatible with the VirtualBox and VMware Fusion providers.

However, they do not (at the time of this writing) provide a Vagrant box up-to-date for use with the libvirt provider.

This repository provides a Packer template that can build such a box.

Requirements⌗

For box creation:

  1. Packer

For launching VMs using the box:

  1. Vagrant
  2. libvirt
  3. vagrant-libvirt

Installation⌗

$ git clone https://gitlab.com/actiniumio/atomic-boxes .
$ cd atomic-boxes

With Makefile⌗

Basic example⌗

$ make all

$ make coreos

$ make photon

Without Makefile⌗

Basic example⌗

$ packer build -force coreos.json

$ packer build -force photon.json
